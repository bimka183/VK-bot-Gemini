# VK-bot-Gemini
Асинхронный VK-бот на vkbottle с интеграцией Gemini AI (Google Generative AI). Поддержка Docker, прокси и управления конфигурацией через переменные окружения.

# VK Gemini Bot 🤖✨

Асинхронный чат-бот для социальной сети ВКонтакте, работающий на базе нейросети **Gemini 2.0 Flash**. Бот умеет отвечать на любые текстовые вопросы, используя современный SDK от Google.

## 🚀 Особенности
* **Асинхронность**: Построен на базе `vkbottle` и `aiohttp`.
* **Gemini AI**: Интеграция с последними моделями Google Generative AI.
* **Docker Ready**: Готовые конфигурации для быстрого развертывания в контейнере.
* **Proxy Support**: Нативная поддержка прокси для обхода региональных ограничений.
* **Безопасность**: Управление ключами через `.env` файлы (секреты не попадут в репозиторий).

## 🛠 Инструкция по запуску

### 1. Подготовка окружения
Склонируйте репозиторий и создайте файл `.env` в корневой папке, используя пример:
```
bash
cp .env.example .env
```
Заполните ваши данные в .env:
```
VK_TOKEN — ваш токен сообщества ВК.

GEMINI_API_KEY — ключ API от Google AI Studio.

PROXY_URL — адрес вашего прокси (например, http://127.0.0.1:10809).```
```
### **2. Запуск через Docker (Рекомендуется)**
Убедитесь, что у вас установлен Docker и Docker Compose. Запустите бота одной командой:

**Bash**
```
docker-compose up -d --build
Примечание для Windows/Mac: Если прокси запущен на хост-машине, используйте http://host.docker.internal:порт в настройках прокси.
```

### **3. Локальный запуск**
Если вы хотите запустить бота без Docker:  

Установите зависимости:  

**Bash**
```
pip install -r requirements.txt
```
Запустите проект:


**Bash**
```
python main.py
```
